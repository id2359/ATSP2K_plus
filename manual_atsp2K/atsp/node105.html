tables

This is another version of the original gtables program. In addition to tabulating data in the same fashion as the gtables, this version has the the following new features:
  • nonphysical states are listed
  • the .lsj file is cleaned from nonphysical transitions
  • .lsj is sorted according to the criteria provided by the user
  • concatenated .l files are accepted and the program prints the spectrum as computed from mchf. Good LS spectrum is frequently an indicator for proper computational model.
  • the energy spectrum can be computed from .lsj files, the only difference is that the Zeeman factors which are contained in the .j files will not be displayed.

Depending on provided input files, tables require the following input:

  • .l files or .j files The program prints the spectrum and prompts the user to input the levels to be erased: 
    .....
--------------------------------------------------------------------------------
     Configuration                 Term  J  Total Energy  Levels 
--------------------------------------------------------------------------------
  1. 2s(2).2p(3)4S3              4S   3/2 -54.554039298                     
  2.                                  3/2 -54.554039298           

  3. 2s(2).2p(3)2D3              2D   5/2 -54.466443168  19224.37           
  4.                                  5/2 -54.466443168  19224.37 
  5.                                  3/2 -54.466403464  19233.09     8.71  
  6.                                  3/2 -54.466403464  19233.09 

  7. 2s(2).2p(3)2P1              2P   1/2 -54.422633792  28839.04           
  8.                                  1/2 -54.422633792  28839.04 
  9.                                  3/2 -54.422633692  28839.07     0.02  
 10.                                  3/2 -54.422633692  28839.07 
--------------------------------------------------------------------------------
B.lsj not found ...

The following duplicate states will be erased:

  2   4   6   8  10 

Select more nonphysical, use a "+" at the end for a cut-off point. Example:
3 8 +, will additionally remove 3, 8 and higher.

    The program will automatically erase all states that are duplicated, or nonphysical (i.e. the same states but with high energy).

  • .lsj files

    If only .lsj files are present the user needs to enter the atomic number and the number of electrons, since this information is not present in the .lsj files.

  • .lsj and .j files If both files are present the user may select the sorting criterion from one of the following: 
     1. Transition energies.
 2. Lower-Upper.
 3. Upper-Lower.
 4. f value.
 5. Line strength.
 6. Wavelength(air).
