I/O Files

bp_mat relies on a set of input files, Figure  9.33. All of them, except <name>.c are in binary format. The output files, hnr.lst.s, hzeta.lst.s, hspin.lst.s contain the nonrelativistic part and the J-dependent contributions to the Hamiltonian. The reason all IO files from serial calculation have the suffix .s, is to distinguish the files generated from nonh. Some of the filenames of the Breit-Pauli calculation bp_ang, bp_mat, bp_eiv coincide with the ons from the nonh, mchf calculation. This becomes is inconvenient when computing in the same directory.
