The subroutine NINE


(33)

The subroutine has the input arguments J1, J2, J3, L1, L2, L3, K1, K2, K3, I and output arguments IN and AA. If the parameter I=1, the subroutine only checks the triangular conditions of a 9j- coefficient. If these are not satisfied, then IN=0, and IN=1 otherwise. At other values of I, the subroutine calculates the value of 9j- coefficient and assigns it to the output parameter AA.
